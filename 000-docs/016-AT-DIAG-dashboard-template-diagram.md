## Component Details

### Header Section
- **Title**: "Dashboard" (text-3xl font-bold)
- **Subtitle**: "Manage your athletes and track performance" (text-zinc-600)
- **Primary CTA**: "Add Athlete" button (top-right, prominent, dark background)

### Athlete Card Structure
Each athlete card contains:

1. **Photo Section** (Square aspect ratio)
   - Athlete photo if uploaded
   - Placeholder user icon if no photo
   - Hover effect: slight scale-up

2. **Info Section**
   - Athlete name (text-xl font-semibold)
   - Position with icon
   - Grade level with icon
   - Team/club name with icon

3. **Stats Preview** (3-column grid)
   - Games played
   - Goals scored
   - Assists (or position-specific stats)

### Grid Layout
- **Mobile**: 1 column
- **Tablet**: 2 columns
- **Desktop**: 3 columns
- **Gap**: 1.5rem between cards

### Color Scheme
- **Background**: zinc-50 (light gray)
- **Cards**: white with zinc-200 border
- **Hover**: zinc-300 border
- **Text**: zinc-900 (headings), zinc-600 (body)
- **Buttons**: zinc-900 background, white text

### Interactive Elements
1. **Add Athlete Button** (Header): Opens athlete creation form
2. **Athlete Cards**: Click to view detailed athlete profile
3. **Empty State Button**: Same as header Add Athlete button
